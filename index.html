Vagrant / Node.js

Vagrant setup to get a Node.js install up and running. This code generate 1 VM.

Requirements

  • Vagrant V1.2.2 min (download and install from http://www.vagrantup.com/downloads.html)
  • Virtualbox V4.2.12 min (download and install from https://www.virtualbox.org/wiki/Downloads)

Add a standard box

vagrant box add precise32 http://files.vagrantup.com/precise32.box

Get the repository

git clone git@github.com:jgrossiord/vagrant-node-hello-world.git

Start the VM

cd vagrant-node-hello-world
vagrant up

This will startup the VM. If you have more than one network card on your PC, it will ask which card you want to bridge to the VM. The VM will have a private IP (10.11.12.2) and a bridge address, taken from your DHCP server, enabling you and your co-workers to connect to the VM's. At the end of the startup process, the URL of the application will be shown.

Other commands (see Vagrant doc)

To suspend the VM

vagrant suspend

To destroy the VM

vagrant destroy

Connect in SSH to the VM

vagrant ssh
